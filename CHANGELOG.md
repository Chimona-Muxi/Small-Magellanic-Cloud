# 更新日志

这个文件记录用户能感知到的变化，以及会影响后续维护判断的结构调整。内部小修不单独写成噪声，除非它改变了维护边界、交互语义或验证方式。

## 2026-07-03

### 设置面板

- 修复语言和外观图标的彩色/灰色纹理残影：设置项图标改为直接 SVG 背景渲染，不再由 emoji 与 mask 叠加生成。
- 同步修复首页、学习页、模块页以及三个棋类游戏页的设置图标规则。
- 统一静态资源版本到 `20260703a`，避免浏览器继续使用旧 CSS/JS。

### 学习工具语言

- 学习工具页、p进数转换器入口页、下载页、网页版转换页补齐页面标题的语言切换。
- 工具名按界面语言本地化：简中“p进数转换器”、繁中“p進數轉換器”、法语 `Convertisseur p-adique`、日语 `p進数変換器`、韩语 `p진수 변환기`、俄语 `Конвертер p-адических чисел`。
- 补齐导航 `aria-label`、设置按钮、语言选择、外观选择等可访问文案。
- 下载页三张平台卡片的说明进入统一翻译字典。
- 网页版 p进数转换器补齐表单标签、转换按钮、示例区标签、空输入、计算中、指令过长和计算失败状态。

### 文档风格

- `README.md` 从简单运行说明改为项目地图，明确站点路径、代码边界、部署方式和维护规则。
- `STYLE_GUIDE.md` 改为 Small Magellanic Cloud 设计宪章，强化页面类型、设置面板、学习工具、棋类游戏和可访问性规则。
- `CHANGELOG.md` 改为发布记录格式，按用户可感知变化分组。

### 既有设计体系整理

- 保留第一性原理、简洁、高级、大气、正式、规范、美观的判断口径。
- 保留 Small Magellanic Cloud 的五个气质词：安静、清楚、深远、克制、有秩序。
- 明确后续 UI 修改优先从减法、对齐和语言体系开始，而不是先加装饰。

## 2026-07-01

### 页面降噪

- 精简首页、游戏页、棋类页的重复信息，去掉重复标题、重复面包屑、卡片标签和卡片底部的重复入口。
- 三个棋类游戏页统一去掉“棋类系统”语义，改为更轻的“棋类”。

### 设置交互

- 语言和外观改为点击右侧当前值区域触发，左侧说明只负责展示。
- 设置面板改为主列表加二级选择页结构，为后续设置项扩展预留空间。
- 提升设置面板背景实体感，避免与游戏侧栏、棋盘或星云背景混在一起。
- 统一桌面端和移动端页眉/设置入口，减少当前页重复标识。

### 联机与安全

- 三个联机棋类游戏继续使用 SSE 实时同步。
- 服务端加入响应头立即 flush 与心跳保活，降低移动端或代理断开长连接造成的同步延迟。
- 修复西洋跳棋、土耳其跳棋联机昵称在玩家卡片、指标和日志区域通过 HTML 字符串渲染导致的 XSS 风险，改为 DOM 节点与 `textContent` 输出。
