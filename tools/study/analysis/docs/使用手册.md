# 数学分析学习辅助器使用手册（中文）

**版本**：v0.30  
**风格**：模仿 GeoGebra CAS 命令格式  
**核心**：纯符号运算（非数值近似）

---

## 1. 简介

本工具用于数学分析理论学习，支持使用类似 GeoGebra CAS 的命令格式进行符号推导。

支持的主要内容：
- 极限
- 导数与偏导数
- 积分（不定、定、广义）
- 级数展开与求和
- 常微分方程
- 代数化简与因式分解
- Gamma / Beta 等特殊函数

---

## 2. 基本用法

```bash
python src/cas_parser.py examples/cas_test.ma
```

输入文件使用 `.ma` 后缀，每行一条 CAS 命令。

---

## 3. 支持的命令

### 3.1 极限
```ma
Limit(sin(x)/x, x, 0)
Limit(1/x, x, oo)
```

### 3.2 导数 / 偏导数
```ma
Derivative(sin(x)*exp(x), x)
Derivative(sin(x), x, 3)          # 三阶导数
```

### 3.3 积分
```ma
Integral(sin(x), x)               # 不定积分
Integral(exp(-x), (x, 0, oo))     # 广义积分
```

### 3.4 级数展开
```ma
Series(sin(x), x, 0, 6)
```

### 3.5 级数求和
```ma
Sum(1/k**2, (k, 1, oo))
```

### 3.6 方程求解
```ma
Solve(Eq(x**2 - 3*x + 2, 0), x)
```

### 3.7 微分方程
```ma
SolveODE(Eq(f(x).diff(x), f(x)), f(x))
```

### 3.8 化简与分解
```ma
Simplify((x**2-1)/(x-1))
Factor(x**2-1)
Expand((x+1)**2)
```

### 3.9 特殊函数
```ma
Gamma(1/2)
Beta(2, 3)
erf(1)
expIntegral(1)        # E₁(1)
expIntegral(2, 1)   # E₂(1)
sinIntegral(1)
cosIntegral(1)
gammaRegularized(2, 1)
betaRegularized(2, 3, 0.5)

# 狄拉克 δ 函数
DiracDelta(0)

# 欧拉 φ 函数（欧拉函数）
totient(10)
eulerPhi(15)

# 狄利克雷 η 函数
DirichletEta(2)
```

### 3.10 扩展功能（GeoGebra 缺失）
```ma
FourierSeries(sin(x), x, -pi, pi, 3)
LagrangeMultiplier(x*y, x + y - 1, x, y)
CriticalPoints(x**2 + y**2, x, y)
ImplicitDerivative(x**2 + y**2 - 1, x, y)
Limit(1/x, x, 0, '+')
```

### 3.11 数列与递推（an / Sn）
```ma
# 斐波那契数列
RecurrenceSolve(a(n) - a(n-1) - a(n-2), a(n), {a(0):0, a(1):1})

# 部分和 Sn 的递推
RecurrenceSolve(S(n) - S(n-1) - n, S(n), {S(0):0})
```

### 3.12 傅立叶分析
```ma
# 傅立叶级数（已有）
FourierSeries(sin(x), x, -pi, pi, 3)

# 傅立叶变换
FourierTransform(exp(-x**2), x, k)

# 逆傅立叶变换
InverseFourierTransform(exp(-k**2/4)/sqrt(2), k, x)
```

### 3.12 绘图功能（GeoGebra CAS 风格）
```ma
# 二维函数图像
Plot(sin(x), x, -pi, pi)

# 隐函数图像
PlotImplicit(x**2 + y**2 - 1, x, y)

# 参数方程
PlotParametric(cos(t), sin(t), t, 0, 2*pi)

# 极坐标
PlotPolar(1 + cos(theta), theta, 0, 2*pi)

# 三维曲面
Plot3D(x**2 + y**2, x, -2, 2, y, -2, 2)
```
