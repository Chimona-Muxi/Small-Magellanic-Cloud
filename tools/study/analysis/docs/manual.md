# 数学分析学习辅助器使用手册

## 支持的符号运算类型

### 1. 极限 (Limit)
- 单变量极限
- 多变量极限
- 单侧极限、无穷极限

### 2. 导数与微分
- 一阶导数
- 高阶导数
- 偏导数
- 全微分

### 3. 积分
- 不定积分
- 定积分
- 广义积分（含无穷限）
- 重积分（未来扩展）

### 4. 级数
- 泰勒级数展开
- 傅里叶级数
- 幂级数
- 级数求和

### 5. 微分方程
- 常微分方程符号求解
- 线性 ODE
- 部分 PDE（未来扩展）

### 6. 其他数学分析内容
- 级数收敛性判别
- 多元函数极值
- 拉格朗日乘子法
- 隐函数微分
- 渐近展开

## 使用方法

```bash
python src/parser.py examples/analysis.tex
```

## 扩展方向

- 更强大的 LaTeX 解析（使用 `sympy.parsing.latex`）
- 支持更多数学分析定理的自动验证
- 可视化输出（matplotlib + sympy）
- 与高等代数学习辅助器联动

当前版本专注于**符号运算**，所有结果均为精确符号表达式。
