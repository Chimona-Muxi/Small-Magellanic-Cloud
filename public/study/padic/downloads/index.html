下载安装器

选择你的系统。安装器会先清理旧版本，再下载 main2.0 分支源码并调用项目内置安装脚本。

安装 / 重装说明

这些文件是未签名的个人项目安装器。浏览器会弹出下载或保存提示；macOS 可能会先拦截直接双击打开，这是系统安全策略，不能由网页绕过。

macOS

如果双击提示无法验证，请在访达中右键文件并选择“打开”，或在终端运行 bash ~/Downloads/install-macos.command。脚本运行后会通过 sudo 请求密码授权。

自动清理旧版本

安装器会先删除旧的 smc-padic、padic_converter 和 .padic_config，再安装 main2.0 最新版。

Linux install-linux.sh

下载后打开或在终端运行，先清理旧版本，再安装 smc-padic。

 macOS install-macos.command

下载后打开，按系统提示允许执行；脚本会请求密码授权。

 Windows install-windows.bat

下载后双击运行，先清理旧安装目录，再执行 Windows 安装流程。
